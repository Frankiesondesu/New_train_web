# 途轨 · 铁路旅行地图

一个无需构建工具即可运行的铁路旅行网站原型，包含：

- 交互式中国地图与铁路线路筛选
- 公开行政区划 GeoJSON 地理骨架，在瓦片服务不可用时仍保留地图轮廓
- 支持系统主题识别、手动切换并记忆选择的暗色模式
- 地图瓦片错误率过高时自动切换到稳定的行政区划底图
- 点击线路查看沿途车站和代表车次
- 点击车站查看所在线路、停靠车次及周边景点、风景和住宿
- 车站、线路、车次统一搜索
- 桌面端侧栏与移动端底部详情面板

## 运行

直接打开 `index.html`，或在当前目录运行自带的零依赖静态服务器：

```powershell
node server.mjs
```

然后访问 `http://localhost:8080`。

地图、字体和图片使用在线资源，需要联网加载。当前线路、车次和旅行信息为产品演示数据，正式上线前应接入铁路时刻、地图与旅游内容服务，并核对授权和实时性。
